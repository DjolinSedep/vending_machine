Использованные команды:
1)git clone URL_репозитория - для клонирования репозитория к себе на компьютер локально
2)git branch - для того чтобы узнать на какой я ветке нахожусь
3)git checkout home_work_25 - для того чтобы перейти на ветку home_work_25
4)git status - для просмотра изменений
5)git add - для добавления изменений
6)git commit -m "msg" - для создания коммита
7)git push - для отправки изменений
8)git checkout master - для того чтобы перейти в ветку master
9)git merge home_work_25 - для слияния веток
